# Gesture-Controlled Media Player 🎮🎵  

This project allows users to **control VLC Media Player using hand gestures**. It utilizes **OpenCV and MediaPipe** for real-time hand tracking and gesture recognition, making media playback seamless and touch-free.  

## 🚀 Features  
✅ **Play/Pause:** Open palm to play, close fist to pause.  
✅ **Volume Control:** Move hand up/down to adjust volume.  
✅ **Seek Control:** Swipe left/right to rewind or fast forward.  
✅ **Real-Time Hand Tracking:** Uses OpenCV and MediaPipe for gesture detection.  
✅ **Lightweight & Fast:** Runs efficiently on mid-range systems.  

## 🛠️ Technologies Used  
- **Python**  
- **OpenCV** (for image processing)  
- **MediaPipe** (for hand tracking)  
- **VLC Media Player** (for media playback)
